* **imbalanced** training [dataset](https://wiki.nci.nih.gov/display/NCIDTPdata/AIDS+Antiviral+Screen+Data) containing 40000+ compounds.  
* Single-task DNN, input: molecular fingerprint, fully-connected layers.
